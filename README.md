Simple E-commerce project using 
  Laravel Backend, 
  Livewire / Blade / Alpine.js Frontend, 
  SQLite Data

## Features

- **User Authentication**: Laravel Breeze with Livewire
- **Product Management**: Browse products with stock levels
- **Shopping Cart**: Add/update/remove items, per-user cart storage
- **Low Stock Notifications**: Automatic email alerts when stock runs low
- **Daily Sales Reports**: Scheduled job sends daily sales summary
- **Responsive UI**: Tailwind CSS with dark mode support

## Tech Stack

- **Backend**: Laravel 12
- **Frontend**: Livewire, Blade, Alpine.js
- **Styling**: Tailwind CSS
- **Database**: SQLite (development)
- **Queue**: Database driver
- **Mail**: Log driver (development)
